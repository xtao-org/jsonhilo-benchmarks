Benchmark results

The results are sorted from fastest (best) to slowest (worst).

Validate

Validate a JSON sample by parsing it without doing anything with the result.

command average time (s) ratio
deno run jsonhilo/empty.js < data/npm.json 0.278 1.000
jq -c --stream empty < data/npm.json 0.428 1.540
node clarinet/empty.js < data/npm.json 0.490 1.763

Count

Count all values in a JSON sample. jsonhilo is tested using both its low- and high- level interfaces.

command average time (s) ratio
deno run jsonhilo/values.js < data/npm.json 0.248 1.000
node clarinet/values.js < data/npm.json 0.484 1.952
deno run jsonhilo/values.high.js < data/npm.json 0.518 2.089
sh jqcount.sh < data/npm.json 0.768 3.097

jq stream

Transform a JSON sample into jq streamed form. jq should be fastest here.

command average time (s) ratio
